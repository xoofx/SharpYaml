# Summary
YAML schema support for scalar resolution and tag handling.

# Remarks
Schemas control how plain scalars are interpreted (for example booleans, integers, null) and how tags are resolved during parsing and deserialization.
SharpYaml targets YAML 1.2 and provides several schema flavors (Failsafe, JSON, Core, and Extended).

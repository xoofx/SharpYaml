# Summary
Top-level APIs for parsing, emitting, and serializing YAML with SharpYaml.

# Remarks
This namespace contains the main entry points for most applications:

- `YamlSerializer` and `YamlSerializerOptions` for object serialization.
- Low-level `Scanner`, `Parser`, and `Emitter` types for token/event based processing.
